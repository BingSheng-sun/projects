## 题目

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 `""`。

**示例 1：**

**输入：**strs = ["flower","flow","flight"]
**输出：**"fl"

**示例 2：**

**输入：**strs = ["dog","racecar","car"]
**输出：**""
**解释：**输入不存在公共前缀。

**提示：**

- `1 <= strs.length <= 200`
- `0 <= strs[i].length <= 200`
- `strs[i]` 如果非空，则仅由小写英文字母组成

## 代码

### 来自评论区（灵茶山艾府）

#### “以不变应万变”

```python
class Solution:
    def longestCommonPrefix(self, strs: List[str]) -> str:
        s0 = strs[0]
        for j, c in enumerate(s0): # 对可迭代对象按照“序列：对象”的方式进行解包，由于我们首先确定了以s0为准，所以只要按照解包后的结果一一对比，直到找到通过次数最多的序列次数
            for s in strs:
                if j == len(s) or s[j] != c: # 这一列有字母缺失或不同
                # 我认为这一句代码是关键，把住了结果关，也就扼住了问题的咽喉
                    return s0[:j]
        return s0
```

#### python3 zip

```python
class Solution:
    def longestCommonPrefix(self, strs: List[str]) -> str:
        lcp = 0
        for col in zip(*strs): # * 先解包，然后再打包，根据列表中最短的 str 的长度作为界限，符合取前缀的要求。
            if len(set(col)) > 1: # 使用 set 函数去除重复元素，如果集合内元素=1，即共有前缀+1
                break
            lcp += 1
        return strs[0][:lcp]
```
