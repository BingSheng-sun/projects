## 题目

给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串 `s` ，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 每个右括号都有一个对应的相同类型的左括号。

**示例 1：**

**输入：**s = "()"

**输出：**true

**示例 2：**

**输入：**s = "()[]{}"

**输出：**true

**示例 3：**

**输入：**s = "(]"

**输出：**false

**示例 4：**

**输入：**s = "([])"

**输出：**true

**示例 5：**

**输入：**s = "([)]"

**输出：**false

**提示：**

- `1 <= s.length <= 104`
- `s` 仅由括号 `'()[]{}'` 组成

### 来自评论（辅助栈法）

```python
class Solution:
    def isValid(self, s: str) -> bool:
        dic = {'{': '}', '[': ']', '(': ')'}
        stack = ['?']
        for c in s:
            if c in dic:
                stack.append(c)
            elif dic[stack.pop()] != c:
                return False
        return len(stack) == 1
```

```python
class Solution:
    def isValid(self, s: str) -> bool:
        dic = {')':'(',']':'[','}':'{'}
        stack = []
        for i in s:
            if stack and i in dic:
                if stack[-1] == dic[i]:
                    stack.pop()
                else:
                    return False
            else:
                stack.append(i)
                
        return not stack
```

## 反思

栈是什么？我不知道，python 中并没有特定的栈的数据类型，所以会用 list 列表来模拟栈。

> 来自豆包
> > Python 没有专门的栈类型，**用列表 `list` 模拟栈是性能最优的选择**，因为：
> >- 列表本质是**动态数组**，内存是连续的
> > - `stack.append(x)`：在列表**尾部**添加元素（压栈），时间复杂度 O(1)
> > - `stack.pop()`：删除并返回列表**最后一个元素**（弹栈），时间复杂度 O(1)
> 为什么尾部操作是 O (1)？
> 动态数组维护了一个指向尾部的指针，压栈 / 弹栈只需要移动指针，不需要移动其他元素，速度极快。
